Prompt

Act as a senior software architect and electrical engineering workflow expert.

Build a highly professional, scalable, offline-first web application for electrical field data collection, specifically designed for power system studies including:

arc flash analysis
short circuit studies
relay coordination
single line diagram preparation
equipment documentation

The application must be designed with enterprise-grade architecture and optimized to handle large datasets, including high-resolution equipment photos.

The system must be future-ready for conversion into a desktop application (.exe) using Electron or similar frameworks.

Core Objective

Create a structured digital survey platform to replace manual electrical field data collection methods.

The system must allow engineers to efficiently capture equipment data during site visits and automatically organize information for engineering studies.

The system must significantly reduce time required for:

arc flash studies
coordination studies
SLD creation
equipment documentation

Primary Workflow

Project creation → equipment selection → data entry → photo capture → repeat → export project

Detailed workflow:

Create new project

Save equipment record

Complete entry
